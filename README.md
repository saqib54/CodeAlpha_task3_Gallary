## ✨ Features

- **Smart Category Filtering:** Instantly filter images by categories (e.g., All, Nature, City, Tech) with smooth enter/exit animations.
- **Custom Lightbox:** Click on any image to view it in a full-screen, distraction-free overlay.
- **Interactive Navigation:** Navigate through the lightbox using on-screen Next/Prev buttons or your keyboard arrow keys.
- **Beautiful Hover Effects:** Deep shadows, zoom-in effects, and elegant text/icon overlays on hover.
- **Fully Responsive:** Uses CSS Grid to automatically adjust the layout for mobile phones, tablets, and desktop screens.

---

## 🛠️ Technologies Used

- **HTML5:** Clean and accessible markup.
- **CSS3:** Flexbox & Grid for responsive design, Backdrop-filter for glass effects, and CSS Transitions/Animations.
- **Vanilla JavaScript:** DOM manipulation, array filtering, and event listeners for the Lightbox and keyboard support.
- **Font Awesome:** For elegant UI icons.
